\begin{itemize}

\item 按6 bit 分段查找（如图\ref{fig1}所示）：
    \begin{itemize}
        \item key中的高6位查找root节点；
        \item key中间6位查找middle层的节点；
        \item key中的低6位查找叶子节点。
    \end{itemize}
\item API接口：http://lxr.free-electrons.com/source/include/linux/radix-tree.h?v=2.6.39

\end{itemize}
